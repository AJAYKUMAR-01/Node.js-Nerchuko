MongoDB
-------
-> No-sql Document based database[sql -> table based]
-> strong support for aggregation pipelines
-> works on BSON format
-> Best for Node applications

Architecture
----------------
-> sql database lo ala aythey tables untayo, mongoDB lo collections untayi.
eg: collections = 'users'

-> Oka collection lo no. of documents untayi

Installation: refer some Youtube Video

--------------------------------
-> Make sure your mongodb is running [else run mongosh command on cmd]
